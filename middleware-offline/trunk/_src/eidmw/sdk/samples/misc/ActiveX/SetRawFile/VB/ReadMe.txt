Object
------
This is a sample for :
- Using the ActiveX with VB.NET
- Showing how to parse an .eid file without the missing SetRawFile function
- Converting Card CodePage to Windows CodePage

Environment
-----------
The project is made with Visual Studio 2005.
The Activex component must be register. (This is done when installing Portugal e-Id MiddleWare 2.6 or 3.5)

ActiveX can be registered with command line : 'regsvr32 pteidlibaxctrl.dll'
And unregister with :'regsvr32 /u pteidlibaxctrl.dll'
